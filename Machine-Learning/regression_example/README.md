# Linear Regression Example (Python)

This project shows a simple implementation of linear regression in Python.
The model is computed manually without using machine learning libraries
such as scikit-learn.

The example models the relationship between years of experience and salary.

## Description
The program:
 Computes the regression parameters (slope and intercept)
 Predicts salary values for new inputs
 Displays a graph with the data points and the regression line

Files

reg.py : Python source code
images/ : Regression result image 
report/ :  PDF explanation 

## How to Run
Install required libraries:

in bash :
pip install numpy matplotlib
python reg.py
